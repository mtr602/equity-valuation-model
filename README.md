Apple Equity Valuation Model (DCF)

This project builds a company valuation model in R. 
It collects financial data, calculates key valuation inputs, 
and estimates intrinsic equity value for apple.

Current components:
- Financial data collection
- Beta calculation for cost of equity
- Data preparation for valuation

Next steps:
- Free Cash Flow calculations
- WACC estimation
- DCF valuation
- Relative valuation
